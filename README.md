# Todo App

A simple yet powerful Todo List application built with modern frontend practices using **React**.

---

## 📌 Overview

This project is a task management application that allows users to create, update, and manage daily tasks efficiently.

It is designed to demonstrate core frontend concepts such as **state management, component structure, and user interaction handling**.

---

## 🚀 Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Filter tasks (All / Active / Completed)
* Persistent data (Local Storage)
* Responsive UI

---

## 🛠 Tech Stack

* ReactJS
* JavaScript (ES6+)
* HTML5
* CSS3

---

## 🎯 Purpose

* Practice React fundamentals
* Understand component-based architecture
* Handle state and user interactions
* Build a real-world mini application

---

## 🧠 Key Concepts Applied

* Functional Components & Hooks (`useState`, `useEffect`)
* State management
* Event handling
* Conditional rendering
* Local Storage integration
* Clean component structure
