To-Do List (React)
Description

Simple To-Do List app built with React for learning purposes.

Focus of this project:

Component structure
State management
Basic UI interactions

No backend is included.

Features
Add task (name + category)
Mark task as done
Delete task (with confirmation)
Filter by status (done / not done / all)
Filter by category
Basic routing (login → main app)
Tech
React
Styled Components
React Router
Project structure
src/
components/ // reusable UI components
pages/ // main screens (Login, Home, etc.)
styles/ // styled-components
utils/ // helper functions (if any)
